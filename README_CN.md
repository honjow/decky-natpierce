# Decky Natpierce
[EN](README.md) | 中文

[皎月连](https://www.natpierce.cn) 的 Decky 插件。

真正的点对点直连，实现内网穿透。

## 安装方法

1. 从 [Releases](https://github.com/honjow/decky-natpierce/releases/latest) 下载最新版本
2. 将 `decky-natpierce.zip` 文件复制到你的掌机设备上
3. 在 Decky Loader 设置中启用开发者模式
4. 从 Decky Loader 安装插件
